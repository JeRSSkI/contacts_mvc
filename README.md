# Контакти (MVC)

Простий приклад застосування шаблону проектування **MVC (Model-View-Controller)** на Python.

## Опис проекту
Цей проект реалізує базову систему контактів, де користувач може:
- Додавати нові контакти (ім'я та номер телефону)
- Переглядати список усіх контактів

Проект демонструє розділення логіки на:
- **Model** – робота з даними (зберігання контактів)
- **View** – відображення інформації та взаємодія з користувачем
- **Controller** – керує логікою програми та координує Model і View

## Файли проекту
- `model.py` – модель даних (контакти)
- `view.py` – відображення та інтерфейс користувача
- `controller.py` – контролер, який об’єднує Model і View
- `main.py` – точка входу в програму
- `README.md` – опис проекту
